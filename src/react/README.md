# React Projects

This directory contains React frontend projects. Each subdirectory is a standalone npm project.

| Project | Description |
|:--------|:------------|
| [`maze_web_server`](./maze_web_server/README.md) | React SPA frontend for the `maze_web_server` REST API |

## Prerequisites

- Node.js 24+

> **Note:** The `maze_web_server` frontend depends on the prebuilt `maze_wasm` Rust package.
> Build it before running `npm install`. See [`maze_web_server/README.md`](./maze_web_server/README.md) for details.

## Setup

Each project manages its own dependencies. See the individual project READMEs for build, test, and development instructions.

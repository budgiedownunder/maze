# `Maze.Wasm.Controls` Assembly

## Introduction

The `Maze.Wasm.Controls` .NET assembly is written in `C#` and provides custom MAUI control classes and definitions for:

- [Icon](./Pointer/Icon.cs)
- [InteractiveGrid](./InteractiveGrid/Grid.cs)
- [Keyboard](./Keyboard/Keyboard.cs)
- [Pointer](./Pointer/Pointer.cs)

## Getting Started

### Setup
To setup the build environment, run the following from the `Maze.Wasm.Controls` directory:

```
dotnet restore
```

### Build
To build the `Maze.Wasm.Controls` assembly, run the following from the `Maze.Wasm.Controls` directory:

```
dotnet build
```

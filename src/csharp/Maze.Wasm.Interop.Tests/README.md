# `Maze.Wasm.Interop.Tests` Assembly

## Introduction

The `Maze.Wasm.Interop.Tests` .NET assembly incorporates the tests for the [`Maze.Wasm.Interop`](../Maze.Wasm.Interop/README.md) .NET assembly.

## Getting Started

### Setup
To setup the build environment, run the following from the `Maze.Wasm.Interop.Tests` directory:

```
dotnet restore
```

### Build
To build the `Maze.Wasm.Interop.Tests` assembly, run the following from the `Maze.Wasm.Interop.Tests` directory:

```
dotnet build
```

### Testing
To run the tests, run the following command from the `Maze.Wasm.Interop.Tests` directory:

```
dotnet test
```

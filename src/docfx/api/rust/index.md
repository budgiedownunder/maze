# Rust APIs

The `maze-project` contains the following `Rust` crates:

| Name | Description |
|--------|-----------|
| [`maze`](../api/rust/maze/view_content.html) | maze definition and calculation library|
| [`maze_console`](../api/rust/maze_console/view_content.html) | maze console application |
| [`maze_wasm`](../api/rust/maze_wasm/view_content.html) | maze web assembly library |
| [`storage`](../api/rust/storage/view_content.html) | maze storage library |
| [`utils`](../api/rust/utils/view_content.html) | utilities library |

# `storage` Crate

## Introduction

The `storage` crate is written in `Rust` and exposes structs, traits and functions for storing data objects.

## Getting Started

### Build
To build the `storage` crate, run the following from within the `storage` directory:
```
cargo build
```

### Testing
To test the `storage` crate, run the following from within the `storage` directory:
```
cargo test -- --test-threads=1
```

### Benchmarking
No benchmarking tests are currently implemented for the crate

### Generating Documentation
To generate and view `Rust` documentation for the crate in your default browser, run the following from within the `storage` directory:
```
cargo doc --open
```

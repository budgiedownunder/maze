# `utils` Crate

## Introduction

The `utils` crate is written in `Rust` and exposes a set of utility functions, via the following modules:
- `file` - utility functions for file-related operations

## Getting Started

### Build
To build the `utils` crate, run the following from within the `utils` directory:
```
cargo build
```

### Testing
No tests are currently implemented for the crate

### Benchmarking
No benchmarking tests are currently implemented for the crate

### Generating Documentation
To generate and view `Rust` documentation for the crate in your default browser, run the following from within the `utils` directory:
```
cargo doc --open
```

# `auth` Crate

## Introduction

The `auth` crate is written in `Rust` and defines the following modules for handling authentication:

- `config` - configuration-related objects
- `hashing` - password hashing and verification

## Getting Started

### Build
To build the `auth` crate, run the following from within the `auth` directory:
```
cargo build
```

### Testing
To test the `auth` crate, run the following from within the `auth` directory:
```
cargo test
```

### Benchmarking
No benchmarking tests are currently implemented for the crate

### Generating Documentation
To generate and view `Rust` documentation for the crate in your default browser, run the following from within the `auth` directory:
```
cargo doc --open
```
